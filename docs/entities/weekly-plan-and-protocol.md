# Недельный план (`WeeklyPlan`) и протокол (`Protocol`)

## Назначение

- **Недельный план** — документ сотрудника на период: задачи вручную и подтянутые из системы (контент-план и др.).
- **Протокол** — документ руководителя: выбранные участники, агрегированные задачи из планов за период.

## Бизнес-правила

- Сотрудник может добавлять/убирать задачи в своём плане.
- Протокол строится по выбранным сотрудникам и их планам (логика на API и фронте `ProtocolsView` / `WeeklyPlansView`).
