# БДР — бюджет доходов и расходов (`Bdr`)

## Назначение

Планирование доходов и расходов по месяцам в разрезе **произвольных статей**; агрегация по кварталам и году на фронтенде.

## Бизнес-правила

- Один документ на **календарный год** (идентификатор и поле `year`).
- Строки хранятся в **JSONB** (`rows`): у каждой строки — имя, тип (`income` / `expense`), суммы по ключам `YYYY-MM`.
- Редактирование и сохранение через API целиком по году (см. фронт `BdrView`).

## Хранение в БД

**Таблица:** `bdr`  
**Модель:** `Bdr` — `apps/api/app/models/finance.py`.

Структура `rows` — массив объектов; точная форма дублируется в TypeScript `BdrRow` в `apps/web/types.ts`.

## API

- `GET /api/finance/bdr?year=YYYY`
- `PUT /api/finance/bdr` — тело `{ year, rows }`

Роутер: `apps/api/app/routers/finance.py`.

## UI

Вкладка **БДР** в модуле финансов (`FinanceView` → `BdrView`).
